WIFI Speed Monitor - About the application

This is a web application created using Python's Flask framework, a simple framework for building web applications using Python. While using a simple sqlite3 database to store the network data. This application utilized the python speedtest module to get the internet speed. The application also utilized socket with socket.io library to continuously update the value shown on the frontend.

The project was originally created to plot charts using Python's matplotlib of my University's wifi speed. But later, I thought, why not just make a new website to display the chart data nicely. The initial project was fun to show my friends just how bad my hostel internet speed was. The image shown as the first one is the monitor on my hostel wifi while the second one is my home network .
